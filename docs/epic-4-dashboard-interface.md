# 史诗4: 核心仪表板界面

## 概述
实现系统的主页仪表板，包括板块热力图、排名列表和市场指数展示，为用户提供直观的市场强度概览。

## 业务目标
- 提供实时的板块强度可视化
- 展示强势板块和个股排名
- 创建直观的市场概览界面
- 建立用户交互基础模式

## 功能范围
- 板块热力图可视化组件
- 板块强度排名列表
- 个股强度排名列表
- 市场强度指数展示
- 基础响应式布局适配

## 验收标准
1. ✅ 热力图正确显示板块强度颜色编码
2. ✅ 排名列表支持实时排序功能
3. ✅ 市场指数计算和显示准确
4. ✅ 界面响应式设计适配不同设备
5. ✅ 数据刷新机制正常工作
6. ✅ 用户交互流畅无卡顿

## 技术约束
- 使用现代数据可视化库（如D3.js或类似）
- 支持实时数据更新
- 响应式设计符合NFR5要求
- 性能满足NFR1和NFR2要求

## 包含功能需求
- **FR1**: 系统应显示带有颜色编码强度等级的板块热力图
- **FR3**: 系统应提供按强度排名的板块和个股列表

## 依赖关系
- 依赖史诗3完成（数据计算引擎）
- 为史诗5提供界面基础

## 风险评估
- **中风险**: 数据可视化性能优化
- **低风险**: 基础UI组件开发

## 预估工作量: 高
