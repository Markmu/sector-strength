## Dev Notes
### 相关架构信息
- **框架**: Next.js 14+ with App Router
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **构建**: Turborepo for monorepo管理

### 项目结构规划
- `web/` - 前端项目根目录
- `web/app/` - Next.js App Router页面
- `web/components/` - 可复用组件
- `web/lib/` - 工具函数和配置
- `web/styles/` - 样式文件
- `web/types/` - TypeScript类型定义

### 核心功能准备
- API数据获取封装
- 错误边界处理
- 加载状态管理
- 响应式布局基础

### 测试标准
- 开发服务器启动测试
- TypeScript类型检查
- 样式构建验证
- 基础组件渲染测试
