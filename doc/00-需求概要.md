# 00. 需求概要文档

## 📋 项目概述
股票市场板块强弱指标系统 - 通过多周期均线系统量化评估股票和板块强弱程度的Web应用。

## 🎯 核心价值功能

### 1. 板块强度可视化
- 热力图展示各板块强度分布
- 颜色编码强度等级（红-橙-黄-绿）
- 实时更新板块强度状态

### 2. 个股强度分析
- 显示个股在多周期均线下的强度表现
- 综合强度得分计算
- 明细展示各均线周期得分

### 3. 强度排名功能
- 板块强度排名列表
- 个股强度排名列表
- 支持按不同周期排序

### 4. 趋势追踪
- 强度变化趋势图表
- 历史强度数据回顾
- 趋势指标显示（↑→↓）

## 👥 用户场景

### 投资者
- 快速识别强势板块和个股
- 发现投资机会和轮动趋势
- 监控持仓股票强度变化

### 分析师
- 分析板块轮动规律
- 研究市场趋势变化
- 生成分析报告

### 交易员
- 寻找交易信号和时机
- 监控市场强弱变化
- 制定交易策略

## 📊 功能优先级

### 🔴 P0 - 必须要有（MVP核心功能）
1. **板块热力图** - 颜色编码显示板块强度分布
2. **个股强度卡片** - 显示综合强度和明细得分
3. **排名列表** - 板块和个股强度排名展示
4. **基础筛选** - 按板块类型、行业分类筛选
5. **时间选择** - 今日、近一周、近一月数据查看

### 🟡 P1 - 重要但可后续迭代
1. **历史趋势图表** - 强度变化折线图功能
2. **自定义配置** - 均线周期和权重调整
3. **实时更新** - 盘中数据自动刷新机制
4. **预警功能** - 强度突破阈值提醒
5. **收藏功能** - 关注列表和个性化设置

### 🟢 P2 - 锦上添花功能
1. **多时间框架** - 周线、月线级别强度分析
2. **对比功能** - 多个股票/板块对比分析
3. **数据导出** - CSV/Excel格式数据导出
4. **移动端适配** - 响应式移动端界面
5. **高级筛选** - 多条件组合筛选
